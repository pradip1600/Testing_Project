# Testing_Project

## Login Page Implementation

This project includes a simple login page with email and password fields. It features:

- Responsive design
- Email validation
- Password validation (minimum 6 characters)
- Clean user interface

### Files:
- `login.html` - Main login page
- `styles.css` - CSS styling
- `script.js` - Form validation and functionality

### How to Use:
1. Open `login.html` in a web browser
2. Enter a valid email address and password (min 6 characters)
3. Click the login button
